# Sales Performance Analysis Dashboard

## Objective
Analyze sales performance data to identify revenue trends, top-selling products, and underperforming regions.

## Tools Used
- Python (Pandas, NumPy)
- SQL (PostgreSQL)
- Power BI
- Excel

## Key KPIs
- Total Revenue
- Monthly Sales Growth
- Top-Selling Products
- Revenue by Region

## Key Insights
- Identified top 5 products contributing to 40% of total revenue
- Found seasonal spike in sales during Q4
- Detected underperforming regions impacting total revenue

## Business Impact
This analysis helps management optimize pricing, inventory planning, and marketing strategy.

## How to Run
1. Load dataset into PostgreSQL
2. Run SQL queries from sql/ folder
3. Run Python notebook for EDA
4. Open Power BI dashboard
